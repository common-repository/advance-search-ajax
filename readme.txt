== Description ==
This plugin to search by Author-Tag-Cats-Date And Keyword with Ajax.
 Loading Ajax Search.
 Eazy To Customize.
Quick and easily access to information.


= Usage =

1. To get started, click on the sub-menu Search Me > Search Setting. 
2. You can use shortcode to run searchform.
2. Example: [Advance_Search_Ajax]

== Installation ==

1. Upload the 'Advance-Search-me' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Add the shortcode to your page.
